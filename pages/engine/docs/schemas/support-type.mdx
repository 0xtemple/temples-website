## Support type

Typescript syntax needs to be followed when writing `defaultValue` in `obelisk.config.ts`.

| Obelisk.config | Move | *Typescript |
| ------- | ------- | ------- |
| "string" | String | string |
| "address" | address | string |
| "bool" | bool | boolean |
| "u8" | u8 | number |
| "u64" | u64 | number |
| "u128" | u128 | number |
| "vector&lt;string&gt;" | vector&lt;String&gt; | string[] |
| "vector&lt;address&gt;" | vector&lt;address&gt; | string[] |
| "vector&lt;bool&gt;" | vector&lt;bool&gt; | boolean[] |
| "vector&lt;u8&gt;" | vector&lt;u8&gt; | number[] |
| "vector&lt;u64&gt;" | vector&lt;u64&gt; | number[] |
| "vector&lt;u128&gt;" | vector&lt;u128&gt; | number[] |
| "vector&lt;vector&lt;u8&gt;&gt;" | vector&lt;vector&lt;u8&gt;&gt; | number[][] |
